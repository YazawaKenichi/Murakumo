#### モータの PWM 制御

DRV8874 の PMODE ピンが GND に落ちているので PN/EN 制御となる。

PN/EN 制御は EN に 0 を入れると無条件でモータが Break する。

PN に 1 を入れて正転、0 を入れて逆転となる。

∴PN には常に 1 を入れ続けて、EN を PWM 制御する
